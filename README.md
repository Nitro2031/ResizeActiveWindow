# ResizeActiveWindow
Resizes the active browser window to a preset size.

Clicking the expansion icon will display a pop-up UI.

Clicking the Settings button will display the preset size settings screen.

You can add or delete preset sizes as you like.

When you press the button for a preset size you've set, the window size will change to the set size.

Added presets will be saved in the browser.


ブラウザウィンドウをプリセットサイズに変更します。

拡張アイコンをクリックするとポップアップUIを表示します。

設定ボタンでプリセットサイズの設定画面を表示します。

プリセットサイズは任意に追加したり削除したりできます。

設定したプリセットサイズのボタンを押すと、そのウィンドウのサイズが設定したサイズへ変更されます。

追加したプリセットはブラウザに保存されます。
